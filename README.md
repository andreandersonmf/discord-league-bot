# Discord League Management Bot

Discord bot made in Python to manage a competitive League server.

## Features
- Transaction requests (add/remove/transfer)
- Roster management per team
- Role sync (Captain/Vice Captain/Player)
- Permission system
- Visual embeds for confirmations/errors

## Tech Stack
- Python
- discord.py
- SQLAlchemy
- SQLite

## How to run
1. Create a `.env` file based on `.env.example`
2. Install dependencies:
   `pip install -r requirements.txt`
3. Run:
   `python bot.py`
